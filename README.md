
# My Game

This is a simple HTML/CSS game created by Kunal Sahu.  
You can play it directly in your browser.

## How to Play
1. Open the link: [Play Here](https://kunalsahu3208-prog.github.io/my-game/)
2. Use arrow keys to move your character.
3. Avoid obstacles and score points.

## Installation
- Clone this repo or download the files.
- Open `index.html` in your browser.
